# Agent Instructions

This is a Go web application template for exe.dev.

See README.md for details on the structure and components.

## Critical Rules

- **NEVER delete user-created trips, photos, or other user data.** If you need test data, create your own trips/stops/photos. Do not modify or remove anything the user built.
- Before any destructive database operation (DELETE, DROP, etc.), confirm with the user first.
- When testing import/export or similar features, use newly created test data — not existing user content.
